# Análise de Grafos e Clustering

Projeto de análise de redes e agrupamento de dados combinando **teoria dos grafos** (NetworkX) com técnicas de **clustering**, com visualização interativa dos resultados.

## Objetivo

Explorar a estrutura de relacionamentos em dados reais: representar entidades como nós e conexões como arestas, identificar comunidades e padrões de agrupamento que não são visíveis em tabelas.

## O que o notebook cobre

- Construção e visualização de grafos com NetworkX
- - Métricas de rede (centralidade, densidade, componentes)
  - - Agrupamento hierárquico e análise de clusters com SciPy
    - - Visualização interativa com Plotly
     
      - ## Tecnologias
     
      - `Python` · `NetworkX` · `SciPy` · `pandas` · `Plotly` · `Streamlit`
