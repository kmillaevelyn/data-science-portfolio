# Dashboard — Mercado de Vagas de Tecnologia (LinkedIn) · PI3

Análise do mercado de trabalho em tecnologia a partir de vagas coletadas do LinkedIn, com **mineração de texto** para extrair hard skills, soft skills e ferramentas mais demandadas — apresentada como dashboard Streamlit + Plotly.

## Objetivo

Responder com dados à pergunta que todo estudante da área se faz: o que o mercado realmente pede? O projeto transforma descrições de vagas em indicadores visuais de competências — e fundamentou a defesa do curso BCTec como formação alinhada à demanda real.

## O que o notebook cobre

- Limpeza e estruturação de dados textuais de vagas do LinkedIn
- Extração de keywords: hard skills, soft skills e ferramentas
- Dashboard Streamlit autocontido com quatro visualizações Plotly
- Fallback com dados simulados quando o CSV não está disponível

## Tecnologias

`Python` · `pandas` · `Plotly` · `Streamlit`
