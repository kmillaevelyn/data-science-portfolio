# Dashboard — Impacto da IA em Estudantes

Dashboard interativo em **Streamlit** que explora dados sobre o impacto do uso de inteligência artificial na rotina e no desempenho de estudantes.

## Objetivo

Transformar um dataset bruto em uma ferramenta de exploração visual: filtros interativos, indicadores-resumo e gráficos que permitem investigar padrões de uso de IA por perfil de estudante.

## O que o projeto cobre

- Limpeza e preparação dos dados com pandas
- Construção de KPIs e visualizações com Plotly
- Interface interativa com Streamlit (filtros, abas e métricas)
- Leitura analítica dos padrões encontrados

## Tecnologias

`Python` · `Streamlit` · `Plotly` · `pandas`
