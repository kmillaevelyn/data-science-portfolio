# Dashboard — Hábitos de Estudo e Saúde Acadêmica

Análise exploratória interativa de um dataset com **31 variáveis** sobre hábitos de estudo, rotina e indicadores de saúde de estudantes.

## Objetivo

Investigar as relações entre rotina de estudo, sono, hábitos e desempenho acadêmico — e transformar essa análise em um dashboard onde qualquer pessoa pode explorar os cruzamentos por conta própria.

## O que o projeto cobre

- Análise exploratória (EDA) das 31 variáveis do dataset
- Cruzamentos entre hábitos, saúde e desempenho
- Visualizações interativas com filtros por perfil
- Leitura crítica dos padrões e limitações dos dados

## Tecnologias

`Python` · `pandas` · `Plotly` · `Streamlit` · `Seaborn`
