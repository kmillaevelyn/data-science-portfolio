# Hidrotech — Sistema de Reaproveitamento de Água

Projeto de **engenharia de produto** desenvolvido em equipe: concepção de um sistema de reaproveitamento de água com análise de viabilidade econômica parametrizada em Python.

## Objetivo

Unir o projeto físico do produto a um **simulador de viabilidade**: dado o consumo, a tarifa local e o custo de implantação, o modelo calcula a economia mensal e o tempo de retorno do investimento (payback).

## Resultados do cenário base

- Economia estimada: **~R$ 20/mês**
- - Payback do investimento: **~44 meses**
  - - Parâmetros ajustáveis (tarifa, consumo, custo) para simular outros cenários
