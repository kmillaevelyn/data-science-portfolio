# Classificação Térmica de Motores Industriais (PI5)

Visão computacional aplicada à **manutenção preditiva**: classificação de imagens térmicas de motores industriais para detecção de anomalias, no contexto da Indústria 4.0.

- **Modelos avaliados:** KNN, Árvore de Decisão, Random Forest, SVM Linear e SVM Polinomial
- **Metodologia:** pré-processamento das imagens térmicas, ajuste de hiperparâmetros com validação cruzada e comparação por matriz de confusão
- **Contexto:** Projeto Integrador 5 (UNIFEI), em equipe com João Vitor Fidanza e Dimas de Barros Cobra
- **Resultado:** Árvore de Decisão liderou com 83,33% de acurácia no teste; ensemble por voto majoritário empatou em 2º lugar com 79,63%
