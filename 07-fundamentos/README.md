# Fundamentos — Primeiros Passos em Python

Onde tudo começou: os primeiros programas em Python, escritos em 2024, preservados aqui como registro honesto da trajetória.

## Por que manter isso no portfólio?

Porque evolução só é visível com ponto de partida. Comparar estes primeiros exercícios com os projetos de Machine Learning das seções 01 e 02 mostra o percurso real: de estruturas básicas de programação a modelos de classificação, NLP e dashboards em produção.
